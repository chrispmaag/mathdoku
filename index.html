How to Play

The Goal

Fill every cell in the grid with a number so that each number appears exactly once in every row and every column — just like Sudoku, but smaller.

  • Easy (3×3): use numbers 1, 2, 3
  • Medium (4×4): use numbers 1, 2, 3, 4
  • Hard (5×5): use numbers 1, 2, 3, 4, 5

Cage Hints

Colored groups of cells are called cages. Each cage shows a hint in its top-left corner — a target number and an operation. The numbers you place in a cage must produce the target when the operation is applied.

HintMeaningExample

Error Highlighting

Cells turn red when a rule is broken — duplicate in a row or column, or a cage whose numbers don't match the hint. Fix the error and the highlight clears automatically.
